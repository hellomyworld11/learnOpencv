# 1. 数字图像处理知识
## 1.图像存储格式
 + 1 用矩阵来表示
  1. 黑白图像(1位) 灰度(8位) 伪彩色(8位) 真彩色(24位)
  2. 伪彩色图像的像素值对应 颜色表的索引，通过索引获取颜色 共256种
# Opencv 
## 2. 图像操作
+ 1.载入
   - imread  加载图像
   - imwrite 写入图像
   - imshow  显示图像
+ 2.图像表示 Mat类(矩阵) 
   ```
     图像矩阵  灰度图像一列只有一个子列就可以(单通道) 
            col0 col1 col2
       row0  0    0     0
       row1  0    0     0
       row2  0    0     0     
    多通道 比如表示RGB颜色模型的矩阵
             col0    col1   col2
       row0 1 0 0   1 0 0    1 0 0
       row1 1 0 0   1 0 0    1 0 0
       row2 1 0 0   1 0 0    1 0 0

   ```
   - 复制和赋值  后的对象都是引用的同一块内存   -  引用计数
   - copyTo  clone()  复制到一块新的内存  独立
   -  Scalar() 传递颜色值  Size表示尺寸 CvtColor 颜色空间转换
